Links para Downloads

Pode ser que ao criar uma página web surja a necessida de disponibliar um arquivo para ser baixado, neste caso usamos a tag <a> fazendo alguns ajustes para que o arquivo possa ser baixado. Veja a seguir como isso é feito.

Baixe aqui o arquivo compactado

Conforme pode ser vist ao verificar o código fonte desta pagina, usamos os atributos 'download' e "Type'. Para o atributo type precisamos consultar qual é o media type de acordo com o tipo de arquivo. Essa informação pode ser encontrada na pagina https://www.iana.org/assignments/media-types/media-types.xhtml. Neste caso usamos o media type "aplication.pd" e "aplication.zip"

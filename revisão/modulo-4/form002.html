Especificidades

Vamos ver algumas especificidades de alguns inputs que tornarão o preenchimento de formulários mais interessante. Nos exemplos a seguir veremos situaçòes em que deveremos preencher os campos com informaçoes especificas, e não conseguiremos inserir qualquer outra informação.

"number"

O input especial "number" tem uma caracteristica interessante, visto torna o campo especifico para o preenchimmento de numeros e os caracteres que acompanham os números como traços, pontos e virgulas.Neste caso não conseguimos colocar letras. Para isso definimos o "type" do input como "number". Veja isso ao observar o código fonte e o exemplo a seguir:

Analisando um pouco mais o formulário acima, percebemos a necessidade de fazer um ajuste no campo de Média, visto que este é um tipo de dado que tem um número limitado de caracteres. Para isso usaremos os atributos, "min" com o valor "0" e "max" com o valor "10". Visto que uma média pode variar em decimos de um número também definimos qual será o passo, para isso usamos o atributo "step", com o valor "0.1".

É claro que a principio pode parecer pouco prático sugerir dados, especialmente se os dados a serem sugeridos se referem a informações condicionais como datas e outros dados que podem não ser constantes. Mas isso pode ser facilmente resolvido por implementar alguns scripts no formulário.

Ainda no campo de periodo letivo podemos permitir que seja incluido não apenas o mês mas tambem a data completa e até mesmo a hora. Isso também pode ser automatizado usando JavaScript. email e tel

Dois elementos que aparecem com muita frequência em formulários é o email e o número de telefone.Para incluir estes campos, nós usamos os inputs "email" e "tel". Assim como nos demais casos, estes inputs permitem alguns controles, mas no caso do input "tel" temos algumas dificuldades para determinar alguns controles. Por isso, será preciso usar uma expresão regular (RegEx) para se obter o resultado esperado. Para isso usamos o controle "pattern"

fieldset

De modo geral, sempre que criamos um formulário, costumammos agrupar informacões semelhantes na mesma área, como por exemplo dados pessoais. Isso torna o preenchimento de formulários mais prático e interessante. Com essa finalidade podemos fazer o agrupamento desses campos envelopando-os com a tag "fieldset" (agrupamento de campos) e incluir a tag "legend" para dar um nome ou título para o grupo de dados.

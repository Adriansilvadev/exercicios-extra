Alinhando textos

Uma das coisas que contribuem para que o conteúdo escrito de qualquer site seja agradável de se olhar é a forma como texto está organizado. Para alinhar os textos de uma pagina usamos a propriedade CSS "text-align", essa propriedade tem os seguintes valore principais; left (esquerda), center (centro), right (direita),justify (justificado).

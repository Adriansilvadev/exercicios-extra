Posicionamentos

Quando trabalhamos com uma imagem com dimensões pequenas podemos usar a propriedade background-position para definir o local em que ela vai ficar posicionada na página. Lembrando que quando uma imagem de fundo tem dimensões menores do que o elemento, por padrão ela será repetida como podemos ver no exdemplo a seguir:

Conforme já aprendemos, podemos interromper a repetição usando a propriedade background-repeat:no- repeat; Usando esta propriedade podemos definir a posição desejada para o elemento; se "left top", "left center","lef bottom", "center top", "center center", "center bottom", "right top", "right center" ou "right bottom". vamos ver como isso fica nos exemplos a seguir

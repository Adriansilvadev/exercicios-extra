Conteúdo local em iframes

Além de usar os iframe para exibir conteudo de outras página web, nós também podemos usar os iframes para exibir conteúdo local. Vejamos no exemplo abaixo como faremos isso. Note que o comportamento do iframe é o mesmo, visto que o que mudou foi apenas a fonte do iframe.

Outra cois que podemos fazer com o iframe é inserir um documento embutido diretamente no código HTML, em vez de referenciar um documento externo. Isso é particularmente útil quando precisamos exibir um conteúdo HTML simples dentro de um iframe, sem a necessidade de criar um arquivo separado. Para isso usamos o atributo srcdoc.veja abaico um pequeno exemplo que ilustra isso:

É claro que podemos fazer muito mais com os iframes, como por exemplo, inserir videos. Na verdade já fizemos uso deste recurso em páginas anteriores, quando consideramos a incorporação de vídeos ao conteudo de uma página web. Vejamos novamente um vídeo incorporado em nossa página, e perceba que o iframe foi o elemento utilizado para esta finalidade.

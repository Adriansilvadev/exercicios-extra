Reproduzindo Audio

Vamos ver como podemos inserir audios em uma pagina web usando HTML.

No código fonte desta página usamos a tag <audio> escolhendo o arquivo através do atributo src="local/nomedoarquivo.formatodoarquivo, e adicionamos a propriedade controls.Desse modo o player de midia apareceu em nossa pagina.

Mas pode haver situações em que o navegador não é compatível. Neste casos podemos disponibilizar outros formatos de aruivos como por exemplo OGG, WAV além de MP3 que o formato que geralmente utilizamos como padrão. Para isso usamos a tag <audio>, porém removemos o parâmetro src=" " e adicionamos a tag <source src="local/nomedoarquivo.formato" type="audio/formatodoarquivo".>

Consulte o código fonte para ver como isso pode ser feito. Abaixo você verá o resultado disso.

Uma observação importante. Evite usar arquivos WAV pois este tipo de arquivo torna seu site pesado.

Novamente, assim como no caso dos arquivos de áudio, podemos inserir arquivos de vídeo de 2 maneiras. Vejamos como podemos fazer isso.

Inserindo vídeos de forma simples

O vídeo abaixo foi inserido de forma simples usando a tag <video>

Conforme podemos notar, o vídeo foi carregado com uma proporçao muito grande, inclusive gerando uma barra de rolagem lateral. Adicionamos a propriedade controls dentro da tag <video> para que os botoes de midia pudessem ser visualizados.

Vamos fazer alguns ajustes e incluir outras propriedades para que o vídeo seja carregado corretamente

Dos vários formatos de vídeo usamos os formatos MP4, M4V, OGV, WEBM, deste o webm é o de pior qualidade.

Conforme pode ser notado acima e também ao consultar o código fonte, foram feitos alguns ajustes para que o vídeo fosse exibido corretamente.

Obtemos este resultado usando as seguintes propriedades adicionais: alem de adicionar outros formatos de vídeo para melhorar a compatibilidade dos navagadores usando a tag < source >, definimos a largura do vídeo usando a propriedade style= "width; 500px", isso fez o vídeo ficar com uma proporçao menor. além disso usamos a propriedade "poster" para adicionar uma capa ao vídeo. A propriedade loop também foi adiconada para que o vidéo seja repetido continuamente.

Neste ponto convem fazer uma observação importante: De modo geral sempre será melhor adicionar vídeos com hospegadem fora do seu servidor, visto que isso pode resultar em alto consumo de banda. A seguir veremos como podemos resolver este problema

Inserindo Pseudo-elementos

Conforme o proprio nome deixa claro os Pseudo-elementos são partes de elementos HTML que podem ser selecionadas e estilizadas usando CSS, mas não correspondem diretamente a um elemento HTML. Assim po inserir elementos em uma págian sem que eles apareçam no HTML. Os pseudo elementos são respresentados assim (::) Veja nos exemplos abaixo alguns pseudo-elementos sendo inseridos.

Aqui está um outro pseudo-elemento, o "::first-line". Esse pseudo-elemento permite que você selecione e estilize a primeira linha de texto dentro de um elemento. É útil para aplicar estilos específicos a cabeçalhos ou outros textos longos.

O quarto pseudo elemento que vamos ciatr é o"::first-letter:". Semelhante ao "::first-line", esse pseudo-elemento permite selecionar e estilizar a primeira letra de um elemento. Como pode ser claramente visto neste parágrafo.

O nosso quinto pseudo elemento é o "::selection". Ele permite que você defina um estilo específico para o texto selecionado, tornando-o mais destacado ou personalizado de acordo com o design da sua página da web

Há vários outros pseudo-elementos, mas por enquento estes já são o bastante. Para concluir vamos usar o pseudo-elemento "::marker" para citar alguns outros pseudo-elementos.

São eles:

  • ::backdrop
  • ::cue
  • ::cue-region
  • ::file-selector-button
  • ::grammar-error (Experimental)
  • ::part()
  • ::slotted()
  • ::spelling-error (Experimental)
  • ::target-text (Experimental)

Como pode ser visto há varios pseudo elementos, mas alguns deles estão em fase experimental. Por isso é sempre bom consultar a tabela de compatiblidade dos navegadores antes de escolher um ou outro pseudo-elemento.

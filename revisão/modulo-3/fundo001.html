Background-image

Iniciamos o módulo 3 trabalhando com a propriedade background-image, que é usada para definir a imagem de dundo dos elementos. Vejamos alguns exemplos .

COR SÓLIDA

GRADIENTE

IMAGEM

Conforme notamos nestes exemplos, podemos definir fundos diferentes para elementos diferentes. Também percebemos que podemos definir um fundo de apenas uma cor, gradiente de cores ou uma imagem de fundo.

Para o elemento de COR SÓLIDA definimos o background color. Para o elemento com o GRADIENTE usamos a propriedade background-image com a função linear gradiente com a direção para baixo (to bottom) e as cores yellow e red. Já no caso do elemento IMAGEM usamos a url para localizar o arquivo de imagem na pasta da página

Por fim, como obviamente você já percebeu, podemos definir um fundo para o corpo da página.Para isso seletamos o body no estilo e definimos qual será o background da página, seja uma cor sólida um gradiente ou mesmo uma imagem de fundo, que é o que ocorre nesta página.

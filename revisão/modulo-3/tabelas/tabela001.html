Trabalhando com tabelas

As tabelas costumavam ser uma ferramenta popular para o layout de páginas da web, mas ao longo do tempo, o uso de tabelas para esse fim foi substituído por abordagens mais modernas e flexíveis, como o uso de CSS e layouts responsivos. No entanto, as tabelas ainda têm um papel importante no desenvolvimento de páginas da web, principalmente para a exibição de dados tabulares, como calendários, agendas, dados de preços, dados de produtos e resultados de pesquisas.

Elementos de tabela

Uma tabela em HTML consiste em vários elementos que ajudam a estruturar e organizar os dados dentro da tabela. Aqui estão os principais elementos que compõem a estrutura de uma tabela em HTML:

<table>: Este é o elemento principal que define a estrutura da tabela em si.

<tr>: Este elemento representa uma linha na tabela e contém as células da tabela.

<th> e <td>: Estes são elementos de célula da tabela. O elemento <th> é usado para os cabeçalhos da tabela, enquanto o elemento <td> é usado para o restante do conteúdo da tabela.

<thead>, <tbody> e <tfoot>: O elemento <thead> é usado para agrupar os cabeçalhos da tabela, <tbody> é usado para agrupar o corpo principal da tabela e <tfoot> é usado para agrupar o rodapé da tabela.

Hierarquia de tabelas

a hierarquia dos elementos de tabela é fundamental para entender como eles se relacionam e como a estrutura da tabela é organizada em HTML. Aqui está a hierarquia dos elementos de tabela:

table (Tabela): Este é o elemento de nível mais alto que define a tabela em si.

thead, tbody, e tfoot (Cabeçalho, Corpo e Rodapé da Tabela): Esses elementos podem ser usados para dividir a tabela em seções distintas, dependendo do conteúdo e do propósito. O thead é usado para conter os cabeçalhos da tabela, o tbody para o conteúdo principal e o tfoot para o rodapé da tabela.

tr (Linha da Tabela): Este elemento representa uma linha na tabela e é usado para agrupar células de tabela.

th (Cabeçalho da Tabela): Este elemento é usado para definir os cabeçalhos das colunas ou linhas da tabela. Normalmente, é usado dentro do elemento thead.

td (Célula da Tabela): Este elemento é usado para definir células regulares de dados na tabela. É usado dentro dos elementos tbody e tfoot.

Personalização de tabelas

Conforme vimos até aqui, a tabela é uma composicão de vários elemnetos HTML, assim para estilizar uma tabela podemos seletar o elementol desejado e fazer os ajustes de bordas, cor altura largura background etc, através da folha de estilos.

Uma propriedade que merece destaque é a propriedade border-collapse que usamos para unir ou separar os elementos da tabela. Veja nos exemplos abaixo como essa propriedade atua:

border-collapse: separate;
A1 A2 A3
B1 B2 B3
C1 C2 C3
border-collapse: collapse;
A1 A2 A3
B1 B2 B3
C1 C2 C3

Tabelas Semânticas

A introdução da semântica no HTML5 trouxe consigo uma abordagem mais estruturada e significativa no desenvolvimento de páginas da web. No contexto das tabelas, a semântica oferece a capacidade de atribuir significado e contexto específicos para os diferentes elementos de tabela, resultando em um código HTML mais legível e compreensível tanto para os desenvolvedores quanto para os motores de busca.

Um dos benefícios da semântica no HTML5 é que os navegadores modernos conseguem interpretar e renderizar elementos semanticamente corretos mesmo que estejam em uma posição diferente no código. Isso significa que elementos como header, footer, article, section, entre outros, serão interpretados da maneira correta, mesmo se estiverem fora de ordem.

Uma observação importante

De acordo com a W3C: os fechamentos das tags tr e td não são obrigatórios, porém recomendados; - Se a hierarquia não for respeitada, os elementos vão aparecer antes ou depois da tabela.

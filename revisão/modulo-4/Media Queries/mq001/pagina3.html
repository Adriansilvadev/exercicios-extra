Media Features

Media features em CSS são condições específicas que você pode verificar em uma media query para aplicar estilos diferentes com base em características específicas do dispositivo ou da tela. Essas características podem incluir largura de tela, altura de tela, resolução, orientação, entre outras.

As media features permitem que você crie layouts responsivos e adapte o estilo de uma página da web com base nas características do dispositivo em que ela está sendo exibida. Aqui estão alguns exemplos de media features comuns:

Width (Largura):

min-width: A largura mínima da tela.
max-width: A largura máxima da tela.

Height (Altura):

min-height: A altura mínima da tela.
max-height: A altura máxima da tela.

Device Width (Largura do dispositivo):

min-device-width: A largura mínima do dispositivo.
max-device-width: A largura máxima do dispositivo.

Device Height (Altura do dispositivo):

min-device-height: A altura mínima do dispositivo.
max-device-height: A altura máxima do dispositivo.

Orientation (Orientação):

orientation: portrait: Retrato (altura maior que a largura).
orientation: landscape: Paisagem (largura maior que a altura).

Resolution (Resolução):

min-resolution: A resolução mínima da tela ou dispositivo.
max-resolution: A resolução máxima da tela ou dispositivo.

Dito de Maneira simples os media features fazem um ajuste fino nas propriedades dos media queries. Observe o código fonte na área da cabeça para notar a aplicação destas propriedades.

Perceba que para a aplicação de media features, adicionamos a propriedade and (orientation: portrait) entre parenteses (), para modo retrato e and (orientation: landscape) entre parenteses (), para modo paisagem

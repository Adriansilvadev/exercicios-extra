Em outras situações, podemos permitir que o susuário selecione apenas uma opção. Nestes casosutilizamos o elemento "radio". O elemento input com o atributo type="radio" é utilizado em formulários HTML para criar botões de opção, também conhecidos como botões de rádio. Esses botões são usados quando você deseja que o usuário escolha exatamente uma opção de um conjunto.

Ao contrário das caixas de seleção input type="checkbox",() os botões de opção input type="radio" () são exclusivos dentro do mesmo grupo, o que significa que apenas uma opção pode ser selecionada. value

Agora vamos considerar um atributo que pode passar despercebidos quando criamos os campos radio e checkbox em formulários, o atributo "value". Embora o atributo value no elemento input type="radio" não seja estritamente necessário para o funcionamento básico dos botões de opção, é uma prática recomendada incluí-lo por algumas razões:

"Transferência de Dados:" Quando um formulário é enviado para o servidor, apenas os valores associados aos botões de opção selecionados são enviados. Se você não tiver um atributo value, não haverá um valor específico para o botão de opção escolhido, tornando mais difícil para o servidor entender qual opção foi selecionada.

"JavaScript e Estilo:" Se você estiver usando JavaScript para manipular os valores dos botões de opção ou aplicando estilos com base nas seleções, ter um valor associado facilita essas operações. O valor pode ser acessado e manipulado de forma mais direta usando JavaScript.

" Semântica e Clareza:"Incluir valores nos botões de opção torna o código mais semântico e claro para outros desenvolvedores que possam revisar seu código. Eles podem entender facilmente quais são as opções disponíveis e como os valores estão sendo usados.

Embora o valor padrão seja o próprio texto dentro da tag label, incluir explicitamente um valor fornece um controle mais preciso e é uma prática recomendada para desenvolvimento web consistente e compreensível. Veja o código fonte para perceber a aplicação deste atributo no exemplo abaixo

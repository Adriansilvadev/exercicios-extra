Aplicando Media Queries

Agora vamos criar algumas Media Queries. Conforme você poderá perceber ao consultar o código fonte desta página, a criação de uma Media Querie na verdade , nada mais é do que a criação de um link para uma folha de estilos com um atributo determinante com o valor de acordo com o tipo desejado, podendo ser:
media="screen" (telas),
media="print" (impressão),
media="speech" Projetado para sintetizadores de fala,
media="projection "Projetado para apresentações projetadas, como slides,
media="baraille" Projetado para dispositivos de leitura braille,
media="embossed" Projetado para dispositivos de impressão tátil,
media="tty" Projetado para dispositivos de teletipo,
media="tv" Projetado para dispositivos de televisão,
ou media="all" (todos os tipos de media)

No código fonte dessa página, verifique a área de cabeça, onde você vai encontrar um link CSS com a propriedade media="screen" e na sequência outro link CSS com a propriedade media="print". Sendo que o primeiro(media="screen") definirá a aparência da página em telas, e o segundo (media="print") definirá a aparência da página para impressão.

Faça novamente o teste de impressão para notar a diferença. Aproveite e veja que após o ultimo parágrafo, não vai aparecer o rodapé mas vai aparecer outra informação que não está disponível nesta tela. Para isso usamos a pseudo elemnto ::after onde inserimos uma mensagem a ser exibida somente na impressão da página. Isso mostra como as Media Queries nos permitem estilizar a página para cada tipo de midia em que a Página será apresentada.

Adicionando cores ao site

"Background-color"

Dando prosseguimento aos nossos estudos sobre CSS, vamos abordar as vária maneiras em que podemos personalizar nossa página com o uso de cores.Conforme já visto nos exercícios anteriores, podemos acrescentar uma cor de fundo à nossa página. Para isso nós utilizamos a propriedade CSS Background-color.

Mas podemos personalizar ainda mais a cor de fundo por adicionar duas ou mais cores criando um efeito chamado degradê. Para isso nós utilizamos a propriedade CSS Background-image.

"Background-image"

A propriedade background-image é usada para adicionar um efeito que chamamos de gradiente ou degradê. É este efeito que está sendo usado neste parágrafo. Para isso definimos a direção do gradiente e as cores que serão usadas. Este é o trecho que usamos para obter este resultado:
background-image: linear-gradient(to right, #2B43ED,#2D7BF7,#33A5E1,#2DE6F7,#2BEDC8);

Neste parágrafo aplicamos a mesma propriedade, mas dessa vez mudamos a direção do gradiente, note que as cores escolhidas são mostradas da esquerda para a direita. Aqui está o trecho do código usado:
background-image: linear-gradient(to lef, #2B43ED,#2D7BF7,#33A5E1,#2DE6F7,#2BEDC8);

Agora veja o mesmo efeito em outra posição. Aqui está o trecho do código usado:
background-image: linear-gradient(to top, #2B43ED,#2D7BF7,#33A5E1,#2DE6F7,#2BEDC8);

Por fim temos o efeito aplicado de cima para baixo. E mais uma vez aqui está o código:
background-image: linear-gradient(to botton, #2B43ED,#2D7BF7,#33A5E1,#2DE6F7,#2BEDC8);

Extraindo gradiente de imagens

Para nos ajudar na tarefa de criar degradês podemos usar algumas ferramentas onlines. Um exemplo delas é a ferramenta da Adobe. Esta feramenta pode ser acessada neste link Nela podemos extrair o gradiente de cores de uma imagem. Isso pode ser util por exemplo, quando precisamos harmonizar os elementos do site com a logo do cliente. Para exemplificar vamos tentar extrair as cores do gradiente da logo do instagram, que pode ser vista abaixo:

Usando a ferramenta extrair gradiente do Adobe Colors, obtemos as seguintes cores representadas pelos codigos hexadecimais: #F9D13C,#F18E53,#F45969,#D32F8E,#9D2AA9

Veja abaixo estas mesmas cores apresentadas em 3 DIVs, cada uma com uma direção do gradiente.

Gradiente da logo do Instagram com posição semelhante à original. Aqui está o código: 
            
                div#primeira{
                    width: 100vw;
                    height: 200px;
                    text-align: center;
                    background-image: linear-gradient(25deg,#F9D13C,#F18E53,#F45969,#D32F8E,#9D2AA9);
                }

Gradiente da logo do Instagram com posição da esquerda para a direita. Aqui está o código: 
            
                div#segunda {
                    width: 100vw;
                    height: 200px;
                    text-align: center;
                    background-image: linear-gradient(to left,#F9D13C,#F18E53,#F45969,#D32F8E,#9D2AA9);
                }

Gradiente da logo do Instagram com posição de cima para baixo. Aqui esta o código: 
            
                div#terceira{
                    width: 30vw;
                    height: 200px;
                    text-align: center;
                    background-image: linear-gradient(to bottom,#F9D13C,#F18E53,#F45969,#D32F8E,#9D2AA9);
                }

Gradiente da logo do Instagram com efeito circular (radial). Aqui está o código: 
            
                div#quarta{
                width:100vw;
                height: 200px;
                text-align: center;
                background-image: radial-gradient(circle,#F9D13C,#F18E53,#F45969,#D32F8E,#9D2AA9);
             }

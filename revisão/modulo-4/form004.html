color

O input type="color" é usada para criar um controle de seleção de cor em formulários HTML5. Esse controle permite que os usuários escolham uma cor usando uma interface específica para a seleção de cores.

Seleção de Cores

Note que as cores iniciais foram, azul verde e vernelho. Isso foi definido atrvés do atributo "value" usando os códigos hexadecimais #0000ff, #ff533a e #15ff11. Lembrando mais uma vez que isso pode ser alterado pelo usuário de acordo com suas preferências.

Embora as siglas de cada estado sejam conhecidas, e até conseguimos vêlas ao consultar o código fonte, elas não ficam visiveis na interface do usuário. optgroup

A tag" optgroup " em HTML é usada para agrupar as opções relacionadas dentro de um elemento "select". Isso é útil quando você tem uma lista de opções que pode ser dividida em categorias ou seções. Vamos aplicar esta tag usando o mesmo exemplo da lista anterior mas desta vez vamos agrupar os estados por região. Mais uma vez consulte o código fonte para notar como o resultado foi obtido.

Estados do Brasil por Região

Conforme pode ser visto, dessa vez os estados estão agrupados por região. Isso foi feito através do envelopamento da tag "option" com a tag "optgroup". Ao selecionar cada uma das 5 listas acima, podemos notar que a região fica em destaque com letras em negrito e não pode ser selecionada na lista. Isso acontece porque o nome da região é na verdade o atributo "label". Lembrando que esse label é diferente do label elemento que utilizamos para associar um rótulo descritivo a um elemento de formulário, como um campo de entrada (input), uma caixa de seleção (select), ou uma área de texto (textarea). Essa associação é feita usando o atributo for na tag label, que deve ter o mesmo valor que o atributo id do elemento de formulário que está sendo rotulado.

textarea

A tag "textarea" em HTML é usada para criar uma área de texto editável em um formulário. Diferentemente da tag input type="text", que é adequada para entradas de texto mais curtas, a "textarea" permite que os usuários insiram texto mais extenso, como em caixas de comentários ou mensagens.

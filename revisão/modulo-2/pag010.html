Trabalhando com modelos de caixa (Box Model)

O Box Model é um conceito fundamental em CSS que define como cada elemento HTML é representado como uma caixa retangular na página da web. Essa "caixa" inclui quatro partes:

Conteúdo: É o conteúdo real do elemento, como texto, imagens ou outros elementos.

Padding (Preenchimento): É uma área ao redor do conteúdo que pode ser preenchida com espaço em branco. É como uma almofada ao redor do conteúdo.

Border (Borda): É uma linha que envolve o conteúdo e o preenchimento. Ela pode ter cores, estilos e larguras diferentes.

Margin (Margem): É uma área vazia ao redor da borda. Ela cria espaço entre o elemento e outros elementos vizinhos.

O Box Model é usado para controlar o tamanho, o espaçamento e o layout dos elementos na página da web. Quando você define larguras, alturas, margens, preenchimentos e bordas em CSS, está trabalhando com o Box Model para criar o design visual da página.

Texto formatado

Em algumas situações específicas podemos precisar exibir o testo com uma formatação especial em que seja preciso respeitar caaracteristicas próprias como o espaçamento e recuos de linhas. O exemplo a seguir ilustre bem isso.

        
// Função para calcular a média de três notas
            function calcularMedia(nota1, nota2, nota3) {
              // Somando as três notas
              var soma = nota1 + nota2 + nota3;
        
              // Calculando a média
              var media = soma / 3;
        
              return media; // Retornando o valor da média
            }
        
            // Variáveis para armazenar as notas
            var nota1 = 7.5;
            var nota2 = 8.0;
            var nota3 = 6.5;
        
            // Chamando a função para calcular a média e armazenando o resultado
            var resultadoMedia = calcularMedia(nota1, nota2, nota3);
        
            // Exibindo o resultado no console
            console.log("A média das notas é: " + resultadoMedia);

O resultado acima pode ser obtido com o uso da tag <pre>

Citação Simples

Ao escrever o conteúdo de uma página pode haver ocasiões em que precisamos fazer uma citação. Neste casos a citação deve estar entre aspas, embora isso possa ser feito simplesmente por inserir "aspas" no texto, para manter a sementica do HTML5 usamos a tag <q>. o resultado pode ser notado no exemplo a seguir.

Citação Composta.

Também pode haver ocasiões em que precisamos fazer citações compostas como pode ser visto no exemplo abaixo.

HTML5, a mais recente revisão da linguagem baseada na web, representa um avanço significativo nas capacidades de criação e entrega de conteúdo, trazendo suporte nativo a elementos multimídia, semântica aprimorada e melhor interoperabilidade, tornando-se um marco crucial no desenvolvimento de aplicações web modernas. -W3C

Como pode ser notado, diferente da simples, a citação composta cria um espaço e quebras de linhas no texto.

Esse resultado visual pode ser obtido usando a tag <blockquote>

Aplicando Estilos CSS

Para tornar nossa página web mais agradável precisamos aplicar algumas propriedades de estilo. É nesse posto que Começamos a fazer uso das CSSs.

Há basicamente três maneiras de aplicarmos estilos ao nosso código, são eles: 'estilo CSS inline', 'estilo CSS interno' e 'estilo CSS externo'. Vamos aplicar cada um deles nas página seguintes. O resultado na pagina web será basicamente o mesmo, mas a diferença poderá ser notada ao observar a estrutura do codigo fonte. Vamos começar pelo mais simples e que deve ser usado com menos frequência, o "estilo CSS inline".

Conclusão

Ao olhar o código fonte do trecho acima você vai notar que o estilo CSS inline não é pratico para fazer muitas alterações visuais em uma página já que isso deixa o código muito poluido, além de não ser pratico para personalizar muitos elementos de uma página web.

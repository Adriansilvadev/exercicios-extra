"Aplicando Estilos CSS "

A ultima tecnica para aplicar estilos em uma página Web, é usando os "Estilos Externos". Esta técnica é a mais prática para a maioria dos casos, visto que com apenas uma folha de estilos podemos aplicar estilos para várias páginas, bastando apenas adicionar o link da folha de estilos em cada página.
